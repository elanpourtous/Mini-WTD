Mini-assistant IA (intégré)

Prise de notes accessible, lecture vocale et dictée expérimentale, pour démontrer un assistant simple dans Mini-WTD.

Cet assistant est chargé dans un cadre dédié pour vos démos :

  • Saisie au clavier d’un texte (compte-rendu, idées, liste de tâches…)
  • Lecture à voix haute avec la synthèse vocale du navigateur
  • Dictée vocale si la Web Speech API est disponible

Raccourcis dans l’assistant : Ctrl+Entrée = lancer la lecture · Échap = arrêter.
